Cores em CSS

Representação das cores

Em CSS, as cores podem ser representadas de várias maneiras. Uma das formas mais comuns é através do uso de palavras-chave, como "red" ou "blue". Também é possível usar valores hexadecimais, como #FF0000 para vermelho ou #0000FF para azul. Outra opção é utilizar valores RGB, que consistem em uma combinação de valores de vermelho, verde e azul, cada um variando de 0 a 255. Há também a opção de utilizar valores HSL, que representam a matiz, saturação e luminosidade da cor. Por fim, também é possível usar a função "rgba()" para especificar a cor e sua opacidade.

Gradiente em CSS

Em CSS, podemos usar a função linear-gradient() para criar gradientes de cor. Essa função permite que você defina a direção do gradiente e as cores que deseja usar. A sintaxe básica é linear-gradient(direction, color-stop1, color-stop2, ...), em que direction é a direção do gradiente, como to right, to bottom, etc. color-stop é a cor que você deseja usar e a localização em que deseja definir a transição. Além disso, você pode usar palavras-chave como transparent, currentColor e rgba() para personalizar ainda mais seu gradiente de cor. Também é possível definir gradientes radiais usando a função radial-gradient().
