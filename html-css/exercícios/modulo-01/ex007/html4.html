O que é Semântica?

A semântica em um documento HTML diz mais sobre o significado do que estética. Sendo assim, algumas tags antes usadas em HTML4, atualmente são obsoletas.

Tags obsoletas

As tags obsoletas são tags que não devem ser mais usadas visando a semântica do documento HTML e as recomendações da w3c. Saiba mais w3c recommendation
